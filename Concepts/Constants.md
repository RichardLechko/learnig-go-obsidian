## Definition

Values that are fixed at compile time

## Restrictions

Can only be:

- Number
- String
- Boolean

## Syntax

```go
const boilingF = 212.0
```

Multiple constants:

```go
const (
    whiteIndex = 0
    blackIndex = 1
)
```
